usando link externos

você pode acessar o repositório público no github

você também pode acessar canal gratuito no youtube

  1. target
    1. "_blank" vai abrir o link em uma nova janela em branco
    2. "_self" vai abrir o link na janela ou frame atual (padrão)
    3. "_top vai desfazer todos os frames e abrir o destino no navegador completo"
    4. "_parent" similar ao uso do _top em uma referemcia a janela mae
  2. rel
    1. "next" indica que o link é para a próxima parte do documento atual
    2. "prev" indica que o link é para a parte anterior do documento atual
    3. "author" indica que é um link para o site do autor artigo atual
    4. "external" indica que é um link para outro site que não faz parte do site atual
      5. "nofollow" indica que é um link para um site não endossado, como um link pago
